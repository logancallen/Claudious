# PROPOSAL — Perplexity Comet Browser + Grok Deep Search Routing Update

**Disposition:** SAFE — external tool routing update; bundle with prior Perplexity items
**Category:** CROSS-PLATFORM / ROUTING

## Rationale
Two external-tool updates worth folding into Logan's research routing:

**(a) Perplexity Comet** — standalone AI browser on iOS, Android, Windows, Mac (March 2026 GA). Context-aware assistant with Deep Research integration. Direct Chrome competitor for research workflows. Pairs with Perplexity Pages (captured 2026-04-14) — persistent browser context plus citable artifacts in one stack.

**(b) Grok Deep Search** — benchmarked at ~10× faster than ChatGPT Deep Research, ~3× more web pages searched. Speed/breadth advantage on rapid macro scans and breaking-news catch-up. ChatGPT DR still wins on depth.

Bundles cleanly with three prior PROPOSED items: 2026-04-12-perplexity-computer-agent, 2026-04-13-perplexity-model-council, 2026-04-14-perplexity-session-memory-pages. Time to do a single subscription/routing review across all four.

## Risks
- **Subscription stack creep.** Perplexity Pro + Comet + Grok Premium + ChatGPT Plus + Claude Max is real money. Need ROI per tool.
- **Grok speed claims are vendor-reported** — independent benchmark would be more reliable. Source is vendor-comparison content, treat with skepticism.
- **Browser context fragmentation:** Comet adds another browser to the desktop. May or may not justify the friction.

## Required Actions (for Logan's review)
1. **Subscription audit:** list current spend on Perplexity, Grok, ChatGPT, Claude. Cut anything below threshold (e.g., $20/mo with <2 high-value uses/week).
2. **Update routing rule** in User Preferences (or wherever external-tool routing lives):
   - Speed-priority research breadth → Grok
   - Citation-grade artifacts → Perplexity Pages
   - Deep-depth multi-step research → ChatGPT Deep Research
   - Structured reasoning + execution → Claude
3. **Comet pilot (Mac):** install for one week of macro/investment research. Decide keep/cut based on whether persistent browser context actually shows up as time saved.
4. **Independent Grok speed verification:** run one comparable Deep Research prompt across Grok and ChatGPT and time both — vendor numbers are not gospel.

## Rollback Plan
Cancel subscriptions; uninstall Comet.

**Recommend:** roll into the existing perplexity-model-council subscription review. Don't make it a separate decision.
